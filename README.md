# بوت لتسجيل الإنجازات

## الهدف من البوت

هدف البوت هو إدارة مجموعة من الجروبات خاص ب التطوير الذاتي والمهنى عن طريق مراقبة الأعضاء أثناء تسجيل الإنجازات الخاص 
بتقدمهم واعطاء نقاط في المقابل لعكس تقدمهم مع حظر الأشخاص الغير مشاركين في تسجيل الإنجازات

## طريقة إعداد وتشغيل البوت

### أولاً تحميل الكود 
روح على شريط الأومر ال CLI في ال IDE الخاص بيك زي مثلا VSCode بعد ما تفتح مجلد خاص ب المشروع وبعدين انسخ الامر ودوس
```bash
git clone https://github.com/hussainnl/ach-bot.git

```
