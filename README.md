# بوت لتسجيل الإنجازات
الصفحة دي خاص عرض الهدف والطريقة تشغيل البوت
بالنسبة لمراحل التطوير كانت ازاي ممكن تشوفها من هنا [رحلة التطوير](readme_files/dev_journey.md) 
## الهدف من البوت

هدف البوت هو إدارة مجموعة من الجروبات خاص ب التطوير الذاتي والمهنى عن طريق مراقبة الأعضاء أثناء تسجيل الإنجازات الخاص 
بتقدمهم واعطاء نقاط في المقابل لعكس تقدمهم مع حظر الأشخاص الغير مشاركين في تسجيل الإنجازات

## طريقة إعداد وتشغيل البوت

### أولاً تحميل الكود 
روح على شريط الأومر ال CLI في ال IDE الخاص بيك زي مثلا VSCode بعد ما تفتح مجلد خاص ب المشروع وبعدين انسخ الامر ودوس

```bash
git clone https://github.com/hussainnl/ach-bot.git
cd ach-bot

```
<img width="1042" alt="VSCode Cli-screenshot" src="/resource/clone_command.png">

### ثانياً تجهيز الملفات المطلوبة للتشغيل

1. ملف ال .env ده اسمو كده نقطة و الenv القيم دي لازم تكون موجوده :

```
#.env file
BOT_TOKEN =       #هنا المفروض يكون التكون الخاص ب البوت بعد ماتعملوا ممكن تبحث وتشوف ازاي بتجيبها


DB_HOST=mysql   #طبعا ده اسم الهوس الداتا بيز البوت مربوط ب  MySql Server ممكن تبحث وتشوف ازاي تثبتوا على جهازك او السرفر وتربطوا ب البوت
DB_USER=db_user  #هنا اسم المستخدم اللي هتدخل بيه في سرفر الداتا بيز
DB_PASSWORD=password #هنا كلمة سر المستخدم 
# ملاحظة المستخدم وكلمة السر خليهم مرتبطين ب الداتا بيز الخاص ب البوت فقط مش ب السرفر كلة للأمن
DB_NAME=db_name  # هنا اسم الداتا بيز اللي اتعملت جو السرفر خاص ب البوت
DB_CHARSET=utf8mb4 #مجرد شكل البيانات في الداتا بيز انسخه زي ما هو
```

### ثالثاً تحميل الحزم المطلوبة للتشغيل

كل المطلوب منك انك تكتب الأمر ده في شريط الأومر نفس زي المرة اللي فاتت فوق:
```bash
pip install -r requirements.txt
```

### رابعاً التشغيل 

كل المطلوب منك انك تكتب الأمر ده في شريط الأومر نفس زي المرة اللي فاتت فوق:
```bash
python src/main.py
```
