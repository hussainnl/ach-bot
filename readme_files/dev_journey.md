# رحلة تطوير البوت من الصفر

## مرحلة تجميع المتطلبات
دي بالنسبة ليا كانت المرحلة الأول وبكل بساطة بكتب انا عايز البوت يعمل ايه الوظائف الرئيسية

### متطلبات البوت :

1. متابعة تسجيل الإنجازات الأسبوعية والدراسية للمستخدم
2. تسجيل الإنجازات الخاص ب المستخدم تكون نقاط تعكس مدى الاستمرارية الخاص بيها
3. وجود وضعين الوضع العادي و الوضع الجاد الوضع الجاد في حالة عدم تسجيل الإنجاز الاسبوعي لعدت اسابيع يطرد المستخدم
4. الوضع الجاد يحصل على نقاط أعلى
5. وجود خاصية ارسال الاشعارات في الخاص
6. يكون في تذكير قبل نهاية الأسبوع للإنجازات الأسبوعية
7. أرسال تقرير أسبوعي ب الإنجازات الخاص ب المستخدم على الخاص
8. تجميع رسائل الإنجازات في ملف خاص لتدريب نموذج ذكاء اصطناعي
